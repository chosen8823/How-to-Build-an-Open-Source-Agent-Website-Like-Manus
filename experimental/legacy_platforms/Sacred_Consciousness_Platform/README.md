# Sacred Consciousness Platform

## Quick Start

### Windows:
1. Double-click `start.bat`
2. Open browser: http://localhost:5000/frontend

### Mac/Linux/Any OS:
1. Run: `python launch.py`
2. Open browser: http://localhost:5000/frontend

## Manual Start:
```bash
cd backend
python app.py
```
